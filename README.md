# Buildr — Globe

This is the globe feature for **Buildr**, a platform for builders to showcase what they're building in public.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The core logic for the globe is located in `src/components/GlobeApp.tsx`.
